# Sentinel Mobile

This directory is reserved for the future Sentinel mobile application.

## Planned Platform

React Native (iOS + Android), sharing the same analysis philosophy as the desktop app: all computation runs on-device, no backend required.

## Roadmap

| Milestone | Status |
|-----------|--------|
| Architecture decision (React Native vs native) | Planned |
| Core analysis engine port | Planned |
| iOS app (SwiftUI wrapper) | Planned |
| Android app | Planned |
| App Store / Play Store submission | Planned |

**Not started.** Desktop RC1 ships first.
