# Archive

This directory contains historical development artifacts that are preserved for reference but are not part of the active documentation.

| File | Description |
|------|-------------|
| `IMPLEMENTATION_LOG.md` | Module-by-module development log (Modules 1–19). Detailed record of every implementation decision, file created, and test count. Useful for understanding *how* each module was built. |
| `DEVELOPMENT_ROADMAP.md` | The original development checklist used during the build phase. Now superseded by [../ROADMAP.md](../ROADMAP.md). |
| `module38-report.html` | HTML validation report from the Module 38 ground-truth validation run. |
| `module39-report.html` | HTML report from the Module 39 trade intelligence lab investigation. |
