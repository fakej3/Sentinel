## What to include in a bug report

The more specific the better:

- Sentinel version (shown in window title or About)
- Operating system and version
- Symbol and timeframe that triggered the issue
- Whether Gemini AI narration was enabled
- Full error message or unexpected output
- Steps to reproduce

## Response time

This is an open-source project maintained in spare time. Issues are checked regularly but there is no guaranteed SLA. Detailed, reproducible reports get resolved faster.
